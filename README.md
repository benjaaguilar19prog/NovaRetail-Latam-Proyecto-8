# NovaRetail-Latam-Proyecto-8

"Explorando factores de comportamiento de una empresa de e-commerce Latino Americana" | Proyecto Sprint-8 | TripleTen

Este repositorio contiene datos de una empresa e-commerce Latino Americana. Cuyo objetivo de la empresa es identificar que factores del comportamiento del cliente estan fuertemente asociados con el ingreso anual; para el cierre del año 2024. Ademas de construir un analisis correlacional que combine las diferentes categorias de variables que los registros ofrecen, para entender asociaciones determinantes en el comportamiento de los clientes.

El Proyecto forma parte del curso de formacion Data Analyst de TripleTen. Sprint 8.
